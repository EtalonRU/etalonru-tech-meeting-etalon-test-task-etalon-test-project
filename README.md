# Тестовый проект на FastAPI

Это тестовый проект, созданный для проверки знаний и навыков кандидатов на технические собеседования. Проект включает базовые API-эндпоинты, написанные на FastAPI, и предназначен для демонстрации различных аспектов разработки RESTful API.
Он такой, какой он есть. Где-то неидеальный и даже, возможно, содержит ошибки.
## Описание

Проект представляет собой простое веб-приложение, которое демонстрирует использование FastAPI для создания RESTful API.
В приложении реализованы следующие возможности:

- Базовая аутентификация пользователей
- Работа с базой данных PosgreSQL
- Логирование запросов
- Валидация входных данных

Основная задача кандидата — улучшить существующий код, исправить баги и добавить новые функциональные возможности.

## Установка и запуск

### Требования

- Python 3.10+
- pip
- virtualenv (рекомендуется)

### Инструкции

2. **Установка зависимостей**

    Рекомендуется использовать виртуальное окружение:

    ```bash
    python3 -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    ```

3. **Запуск приложения**

    После установки всех необходимых пакетов запустите контейнеры приложения:

    ```bash
    docker-compose -f docker-compose-local.yaml up -d
    ```

    Запустите само приложение:

    ```bash
    cd src
    alembic upgrade head
    uvicorn main:app --reload
    ```


### Команды

- `uvicorn app.main:app --reload`: Запуск сервера с автоматической перезагрузкой при изменении файлов
- `alembic upgrade head`: Применить миграции
